.. image:: /_static/images/xccdf-logo.png

XCCDF - The Extensible Configuration Checklist Description Format Python Library

This Library provides the means to create, edit or transform an XCCDF XML file in any way you want.

The specification used is the 1.2 Release. More information `here <http://scap.nist.gov/specifications/xccdf/#resource-1.2>`_
